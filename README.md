# TachyonLabs Innovations Pvt. Ltd. 

## How to install

1. Fork and then clone this repository:

```bash
git clone git@github.com:<your_username>/tachyonlabs-in.github.io.git
```

2. Go into project

```bash
cd tachyonlabs-in.github.io
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
